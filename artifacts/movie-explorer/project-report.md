# Movie Recommendation Explorer

## Title

Movie Recommendation Explorer

## Objective

Build a browser-based recommendation engine that suggests similar movies from
metadata alone and makes the scoring logic understandable to a user.

## Process

1. Selected a compact public movie metadata dataset.
2. Normalized genre, year, runtime, director, cast, and plot data for each title.
3. Built a weighted similarity system in JavaScript using genre overlap, release
   era, runtime proximity, director match, actor overlap, and plot keyword overlap.
4. Added filters, recommendation cards, explanation chips, and comparison views.
5. Packaged the entire experience as a static project that works on GitHub Pages.

## Tools

- JavaScript
- JSON data and configuration
- HTML
- CSS
- Browser-side similarity scoring
- GitHub Pages compatible static hosting

## Value Proposition

This project demonstrates that I can design a recommendation workflow, turn
metadata into explainable similarity rankings, and build a polished interactive
experience without relying on a backend service.

## Key Features

- interactive movie selector
- candidate filters for genre, decade, and runtime
- ranked recommendations with explainable match reasons
- comparison charts for similarity drivers, genre distribution, and release years
- downloadable data, config, and source files

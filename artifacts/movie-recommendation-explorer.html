Title

Movie Recommendation Explorer

Objective

Build a static, browser-based recommendation engine that recommends similar movies from metadata alone and presents the reasoning behind each recommendation clearly.

Process

I selected a public movie metadata dataset, normalized its genres, runtime, year, director, cast, and plot text, then designed a weighted similarity system with interactive filters and recommendation breakdowns.

Tools

JavaScript, JSON data, browser-side similarity scoring, HTML, CSS, and GitHub Pages compatible static hosting.

Value Proposition

This project demonstrates frontend engineering, explainable similarity ranking, and the ability to turn raw metadata into an interactive, user-facing recommendation experience.

Deliverable Type

Interactive dashboard, downloadable movie dataset, configurable scoring weights, JavaScript application logic, and styling assets.

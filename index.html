Robot Sensor Data Classification

A robotics-adjacent sensor classification project using smartphone inertial data. It compares lightweight classifiers across six movement states and exports charts, summaries, and reusable project files.

Objective

Classify motion and posture states from inertial sensor features using a public benchmark dataset.

Process

Selected accelerometer and gyroscope features, preserved the official train/test split, and evaluated multiple models.

Tools

PowerShell, CSV processing, SVG chart generation, HTML, CSS, and Markdown.

Value proposition

Shows how I connect machine learning workflows to sensor-heavy, robotics-relevant problems.

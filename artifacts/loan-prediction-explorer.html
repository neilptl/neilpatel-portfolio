Title

Interactive Loan Prediction Explorer

Objective

Create a browser-based lending explorer that lets a user manipulate application inputs, inspect a live approval score, and connect model behavior to patterns in a public dataset.

Process

I selected a public loan approval dataset, modeled key numeric and categorical features with a client-side scorecard, designed a dashboard around scenario testing, and added charts plus nearest-neighbor loan matches to make the output more interpretable.

Tools

JavaScript, Plotly.js, Papa Parse, JSON configuration, HTML, CSS, and GitHub Pages compatible static hosting.

Value Proposition

This project demonstrates frontend engineering, interactive data storytelling, and my ability to turn model logic into a usable browser-side experience instead of a static report. That matters when teams need decision-support tools that make model behavior easier to inspect, discuss, and trust.

Best Fit Roles

Relevant for frontend, full-stack, product engineering, and junior machine learning roles focused on user-facing AI features, explainability, or internal analytics tools.

Why It Matters

In lending and similar decision workflows, stakeholders need to inspect why a case looks strong or weak. This explorer turns scoring logic into something users can question, compare, and discuss.

Deliverable Type

Interactive dashboard, downloadable source dataset, browser-side model configuration, JavaScript application logic, and styling assets.
